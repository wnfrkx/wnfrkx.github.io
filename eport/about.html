PROJECT DESCRIPTION

The main purpose of this website is to be submitted as a group project for our teacher’s requirements. The secondary purpose is to be an interesting way to learn the content within the site and be easily accessible to all.

By closely coordinating with one another and laying out plans to finish, we were able to build a website using our collective skills in website building, graphics design, and note taking during classes for the main body of the content.
